MSKey: A Virtual Meetei Script Keyboard

Type Meetei Script on the go, any device, everywhere. No external app nor installation required.

Enter text below. Drag the dashed wedge in the bottom-right of the text box to change the size of the text box.

Click on "Copy" button to copy the text you have typed. Now you can paste it to anywhere you like!

This functionality is most useful on desktop devices when you have no access to physical (or app) Meetei Script keyboard.
